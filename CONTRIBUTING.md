## How to contribute

#### **Did you find a bug?**

* **Ensure the bug was not already reported** by searching on GitHub under [Issues](https://github.com/andreavalsesia/upo-latex-thesis-template/issues).

* If you're unable to find an open issue addressing the problem, [open a new one](https://github.com/andreavalsesia/upo-latex-thesis-template/issues/new). Be sure to include a **title and clear description**, as much relevant information as possible, and if applicable a **code sample** demonstrating the expected behavior that is not occurring.

### **Improvements to this template are welcome!**

#### **Did you write a patch fixing a bug or adding a new feature?**

* Open a new Pull Request.
* Ensure the PR description clearly describes the problem and solution.
